# 照片管理应用 (PhotoManage) - 项目总结

## 项目概述

这是一个基于 SwiftUI 开发的 iOS 照片管理应用，旨在帮助用户以更直观、高效的方式管理和整理照片库中的照片。应用采用了现代化的卡片式交互设计，提供了类似 Tinder 的滑动手势操作体验。

## 技术栈

- **开发语言**: Swift
- **UI 框架**: SwiftUI
- **照片框架**: Photos Framework (PHAsset, PHPhotoLibrary)
- **最低支持版本**: iOS 15.0+
- **设计模式**: MVVM (Model-View-ViewModel)

## 核心功能

### 1. 照片库访问与权限管理
- 自动请求相册访问权限（读写权限）
- 支持有限访问模式（Limited Access）
- 权限被拒绝时提供 Mock 数据演示

### 2. 主页轮播视图 (HomeView)
- **卡片轮播**: 水平滚动浏览照片组，采用分页式滚动行为
- **动态背景**: 自动提取当前照片组的第一张照片作为模糊背景
- **分组展示**: 默认显示 3 个照片组，每组 15 张照片
- **刷新功能**: 支持手动刷新重新随机加载照片

### 3. 照片浏览与操作 (BrowsingView)
- **卡片堆叠**: 显示前 3 张照片的堆叠效果，支持流畅的过渡动画
- **滑动手势操作**:
  - ⬆️ 向上滑动：删除照片
  - ⬇️ 向下滑动：收藏照片
  - ⬅️ 向左滑动：下一张（跳过）
  - ➡️ 向右滑动：上一张（回退）
- **动画效果**:
  - 卡片跟随手势实时移动和旋转
  - 释放后飞出动画
  - 弹性恢复效果
- **时间显示**: 底部工具栏显示照片拍摄时间（相对时间格式）

### 4. 照片管理功能 (PhotoManager)
- **照片加载**:
  - 从相册随机抽取最多 45 张照片
  - 支持无排序、随机索引访问
  - 自动分组管理
- **收藏功能**:
  - 标记照片为收藏状态
  - 同步更新到系统相册
- **删除功能**:
  - 待删除队列：滑动的照片先加入待删除列表
  - 批量删除确认：用户可以预览待删除照片后确认删除
  - 取消删除：支持取消待删除的照片
  - 真正删除：从 iCloud/系统相册中永久删除
- **撤销功能**: 将底部的照片重新移到顶部

### 5. 删除汇总视图 (DeletionSummaryView)
- 预览所有待删除的照片
- 确认删除或取消删除操作
- 实时更新系统相册

## 数据模型

### Photo 模型
```swift
struct Photo: Identifiable {
    let id: UUID
    let imageURL: URL?          // Mock 数据 URL
    let asset: PHAsset?         // 真实相册资源
    let creationDate: Date      // 拍摄日期
    var isFavorite: Bool        // 收藏状态
    let color: Color            // 占位颜色
}
```

### PhotoGroup 模型
```swift
struct PhotoGroup: Identifiable {
    let id: UUID
    var title: String           // 分组标题
    var photos: [Photo]         // 照片数组
}
```

## 设计特点

### 1. 视觉设计
- **深色主题**: 黑色背景，适合照片展示
- **玻璃态效果**: 使用 `.ultraThinMaterial` 创建磨砂玻璃效果
- **圆角设计**: 大圆角卡片 (32pt)，符合现代 iOS 设计语言
- **阴影层次**: 多层阴影增强立体感

### 2. 交互设计
- **手势驱动**: 完全基于手势操作，无需点击按钮
- **实时反馈**: 手势过程中卡片跟随移动，提供即时视觉反馈
- **流畅动画**: 使用 SwiftUI 动画系统，过渡自然流畅
- **分页滚动**: 横向滚动自动对齐到卡片中心

### 3. 用户体验
- **预览机制**: 删除前可预览待删除照片列表
- **撤销支持**: 支持回退操作
- **空状态提示**: 照片组清空后显示完成状态
- **模糊背景**: 自动切换背景模糊图，增强沉浸感

## 权限要求

- **NSPhotoLibraryUsageDescription**: 需要访问相册以管理和整理照片
- **读写权限**: 支持读取和删除照片

## 工作流程

1. **启动应用**:
   - 检查相册权限
   - 授权后随机加载 45 张照片，分为 3 组

2. **浏览照片组**:
   - 在主页横向滚动选择照片组
   - 点击卡片进入浏览模式

3. **整理照片**:
   - 向上滑动删除不需要的照片
   - 向下滑动收藏喜欢的照片
   - 向左滑动跳过（不处理）
   - 向右滑动查看上一张

4. **确认删除**:
   - 返回时查看待删除列表
   - 确认后从系统相册永久删除
   - 或取消删除操作

## 技术亮点

1. **SwiftUI 高级特性**:
   - `scrollTransition` 实现卡片缩放和 3D 旋转效果
   - `scrollTargetBehavior` 实现分页滚动
   - `scrollPosition` 追踪当前活跃的卡片

2. **Photos Framework 深度集成**:
   - 使用 `PHAsset` 直接访问相册资源
   - `PHAssetChangeRequest` 实现真正的删除和收藏
   - 权限状态管理

3. **MVVM 架构**:
   - 业务逻辑与 UI 解耦
   - 使用 `@StateObject` 和 `@ObservedObject` 管理状态
   - 响应式数据绑定

4. **自定义视图组件**:
   - 可复用的 `AssetImageView` 支持本地和网络图片
   - 自定义卡片组件支持动态布局

## 总结

这是一个设计精良、交互流畅的照片管理应用，充分利用了 SwiftUI 和 Photos Framework 的能力，为用户提供了一种全新的照片整理体验。通过手势操作和视觉反馈，让照片管理变得轻松有趣。项目结构清晰，代码质量高，具有良好的可维护性和扩展性。
